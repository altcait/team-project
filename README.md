# Team Project
## Group TUT5101-BA3185-10
## Domain: Travel Planning

### Team Name: Maple Matrix


A travel planning application that allows users to search for travel destinations (countries) based on various criteria and save destinations to their “favourites” list.

### User Stories
1. As a user, I want to create/access/profile my account so that I can securely access and manage my personalized travel destinations.
2. As a user, I want to save countries so I can easily find them later.

User story 3: As a user, I want to view a list of my saved countries so that I can pick up where I left off, with options to search, delete, and edit details.

User story 4: As a user, I want to search for countries that speak a particular language so I can narrow my list based on my travel preferences.

User story 5: As a user, I want to search for countries in a particular region of the world so I can narrow my list based on my travel preferences.

User story 6: As a user, I want to search for countries that accept a specific currency so I can explore new destinations.

### API
RESTful API (https://restcountries.com/)
